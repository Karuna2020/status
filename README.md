## 👩‍💻 Development

Create a new incident:

```bash
npm run new-incident
```

Update an incident:

```bash
npm run update-incident
```

Delete an incident:

```bash
npm run delete-incident
```

Start a local development server:

```bash
npm run dev
```

Build for production:

```bash
npm run generate
```
